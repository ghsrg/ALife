# Критерії готовності MVP

Проєкт можна вважати придатним до першої симуляції, якщо:

* існує світ;
* працює Tick;
* працюють ресурси;
* працюють матеріали;
* працює енергія;
* існує одна клітина;
* клітина може:

  * жити;
  * рости;
  * синтезувати матеріали;
  * ділитися;
  * помирати.
