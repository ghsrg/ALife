# examples

> Ілюстративні сценарії для читання разом із Canon-документацією.

---

# Призначення

`docs/examples/` містить приклади, які допомагають зрозуміти правила.

Ці файли не створюють нових правил. Якщо приклад суперечить Canon-документу, правильним вважається Canon.

---

# Файли

- `biology-examples.md` — приклади клітин, Joint, lifecycle, communication, organism-like структур.
- `genetics-examples.md` — приклади Genome Runtime, mutation, inheritance, recombination, HGT.
- `config-examples.md` — приклади world/resource/material/field/reaction config.
- `engine-examples.md` — приклади scheduler, ECS, storage, serialization, trace/debug.

---

# Правила

- Приклад не є вимогою.
- Приклад не може вводити новий механізм.
- Приклад має посилатися на існуюче Canon-правило або демонструвати його.
- Довгі приклади мають жити тут, а не в Canon-файлах.
