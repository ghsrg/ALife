# 🧬 ALife Simulation

**ALife Simulation** — це 3D-симулятор штучного життя, створений на основі принципів емерджентності. Проект моделює еволюцію та поведінку складних організмів "знизу-вгору", спираючись виключно на фізику, хімію та базову генетику.

## 🎯 Задачі проекту
Головна мета проекту — створити середовище, де складні форми життя, органи та соціальна поведінка не програмуються заздалегідь, а виникають природним шляхом як інструмент для виживання. Ми досліджуємо, як з базових хімічних речовин та передачі електричних зарядів еволюція здатна "виростити" аналоги нейромереж, м'язів та органів чуття.

## 💡 Основні принципи
1. **Абсолютна емерджентність:** У коді проекту не існує понять "око", "м'яз", "мозок" чи "нейрон". Усі ці структури є наслідком просторової топології клітин, їхнього хімічного складу та взаємодії.
2. **Єдина універсальна клітина:** Відсутні жорстко задані "типи" клітин. Кожна клітина (Part) є універсальним контейнером, властивості якого визначаються лише тим, які речовини вона синтезувала згідно зі своїм геномом.
3. **Енергія як єдина валюта:** Будь-яка дія (ріст, передача сигналів, скорочення) вимагає енергії. Виживають лише ті структури, чия ефективність перевищує витрати метаболізму.
4. **Навчання як хімічний процес:** Навчання не є зовнішньою абстракцією. Воно реалізується через зміну фізичної провідності з'єднань (Joints) або хімічної чутливості клітин (Intrinsic Plasticity) в результаті їхнього використання під час життя.
